## Installation

### Dependencies
- SDL2
- SDL2_image
- gphoto2

Ubuntu
```
apt-get install libsdl2-dev libgphoto2-dev
```
Mac
```
sudo port install libsdl2-dev libsdl2-image-dev libgphoto2-dev
```
### Building
```
make && make run
```
